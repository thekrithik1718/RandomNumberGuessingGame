# RandomNumberGuessingGame

A simple beginner-friendly Number Guessing Game built using HTML and JavaScript.

The program generates a random number between 1 and 100, and the player tries to guess the correct number. After each guess, the game provides feedback such as "Too High" or "Too Low" until the correct number is guessed.

Live Demo : random-number-guessing-game-kk.netlify.app

Tech Stack
- HTML
- JavaScript
